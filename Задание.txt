Дано:
- Список регионов;
- Список городов с привязкой к регионам;
- Список улиц с привязкой к городам;

Задачи:
- Нужно создать БД и заполнить ее тестовыми данными на основе блока "Дано";
- Нужно вывести список адресов на странице;
- Добавить одно текстовое поле, с помощью которого можно искать среди названий регионов, городов и улиц.

Что необходимо использовать:
- Проект должен быть написан на Laravel;
- В качестве базы данных использовать PostgreSQL / - MySQL;
- Для создание и заполнения базы данных использовать миграции и сиды;

Будет плюсом:
- Использовать архитектуру Controller -> Service -> Repository -> Model;
- Развернуть с помощью Docker / Docker Compose;
- Использовать VueJS.
